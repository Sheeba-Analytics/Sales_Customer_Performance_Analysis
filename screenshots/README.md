
# Dashboard Screenshots

This folder contains screenshots of the Power BI dashboard created for the Market Analysis project.

The screenshots provide a visual preview of the key dashboard pages, including:

- Executive summary with key performance indicators
- Customer demographics analysis
- Sales performance by region and product
- Time-series analysis of sales trends
- Product-level drill-through insights

These screenshots help demonstrate the design and analytical insights of the dashboard without requiring Power BI to view the full report.
